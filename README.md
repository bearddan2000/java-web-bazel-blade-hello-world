# java-web-bazel-blade-hello-world

## Description
A POC for blade framework rendering html page.

## Tech stack
- cors

## Docker stack
- l.gcr.io/google/bazel:latest

## To run
`sudo ./install.sh -u`
Available at http://localhost

## To stop
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
